My dear friends and team mates, as I prepare to embark on a new journey, I find myself reflecting on the incredible adventure we've shared over the past 12 years at CBA. It feels like a recap of an unforgettable TV show:

Episode I
The War of Mobile (2012-2013)

In the era of mobile apps, I reverse-engineered the existing Kaching web app, documented the API interfaces and sequence flows so I can effectively guide our iOS and Android engineers in building the first native CommBank app within 6 months.

Episode II
The Rise of Helix (2013-2015)

If you don't know what Helix is, consider yourself lucky. During this time, we built numerous Helix apps and I created the well-known AUTHHACK app to help engineers test Helix apps easily. One of the craziest endeavors was working with David Hua to build the cards view for the tablet CommBank app in 2 sprints.

Episode III
The Microservice Crusade (2013-2017)

We built microservices to decommission the monolithic SocialBanking API, covering accounts, cards, payments, transaction history, and more. This period taught us the importance of designing resilient into microservice system.

Episode IV
The Age of Containers (2017-2020)

Collaborating with AppInfra on K8s, we containerized .NET Framework APIs using Windows base images. We also designed and implemented a codified CD pipeline supporting blue/green releases in production.

Episode V
The Azure Rebellion (2020)

We designed the new overnight batch payment system - "NB02", using Azure Storage Queue and Azure Function, replacing the legacy Java system.

Episode VI
The Kafka Chronicles (2021-2022)

We designed and implemented an event-driven home loan application lodgement system using Kafka.

Episode VII
The AWS Ascension (2022-2024)

Our HomeBuying squads worked hand in hand to release many projects to AWS cloud through DHP. I implemented Lambda, ALB/NLB sticky session and DMS on DHP, and established best practices for hosting .Net Core APIs on Lambda.

....

It has been an honor and a privilege to work with all of you, my friends and team mates. I will miss it dearly. Now, I am ready to depart the fleet and start another adventure. Wish me luck, and....

May the Coding Skills be with you!
